# AI Manga Drama Preproduction

> 将小说、文章、大纲或故事创意转换为专业 AI 漫剧前期制作包 —— 从剧本改编、分镜设计、角色造型到摄影灯光，一站式 preproduction skill。

## 这是什么？

**AI Manga Drama Preproduction** 是一个面向 AI 短剧/漫剧创作者的前期制作 skill。它模拟了一个完整的影视前期制作团队，将文字素材转化为可直接用于 AI 生成的可拍摄剧本和视觉规范。

## 核心角色

Skill 以三个协作部门的方式运作：

| 部门 | 职责 |
|------|------|
| **导演/编剧** | 提取原素材的故事引擎、设计短剧结构、编写高精度 AI 生成提示词 |
| **选角/角色设计** | 将角色转化为可定型的演员形象卡，包含外貌、气质、声音、服装及 AI 连续性锚点 |
| **摄影/灯光/美术** | 定义每镜的景别、角度、运镜、焦段、光圈、快门、灯光方案及美术方向 |

## 工作流程

1. **读取素材** — 识别主角、冲突、欲望、秘密、赌注、基调与受众承诺
2. **选择格式** — 竖屏 9:16（手机短剧）或横屏 16:9（电影感）
3. **构建戏剧骨架** — 前 3 秒爆点钩子 + 中段可见冲突 + 结尾反转/悬念
4. **生成导演-编剧提示词** — 可直接粘贴给 LLM 生成/修改剧本
5. **创建制作剧本** — 带时间码的分场剧本表
6. **创建角色形象表** — 含 AI 连续性提示词的角色设定卡
7. **创建视觉圣经** — 角色、场景、色彩、风格的一致性规范
8. **创建分镜计划** — 每镜包含拍摄参数与 AI 画面/视频提示词
9. **创建声音设计** — 对白、环境音、动作音效、音乐、静音、转场
10. **质量检查** — 连续性、可拍性、反转铺垫与回收

## 核心设计理念

### 分镜三要素原则

每个镜头必须通过三重检查：

```text
镜头怎么拍 + 镜头里有什么 + 观众看到什么
```

- **怎么拍** — 景别、视角、运镜决定情绪引导和信息揭示
- **里有什么** — 角色、场景、台词、音效决定可拍摄元素
- **看到什么** — 观众实际接收的叙事信息

### 角色选角逻辑

```
演员选择 = 角色功能 + 表演气质 + 外貌辨识度 + 声音质感 + 服化道连续性 + AI 一致性锚点
```

不同角色类型的选角侧重：
- **主角** — 共情点 + 可见欲望 + 被压迫但不软弱的身体语言
- **反派** — 压迫感 + 表情管理 + 镜头可放大的细节
- **隐藏身份角色** — 两套形象设计：误导表层 vs 反转真实

### 声音设计原则

- 前 3 秒可用声音爆点：尖叫、耳光、破碎声、心跳、耳语
- 反转时刻需独立声音提示
- 画外音制造悬念
- 静音本身也是声音设计
- 每镜一个主导音频意图

## 爆点 & 反转模式

**钩子类型：** 危险、矛盾、背叛、地位翻转、秘密物品、倒计时

**结尾类型：** 身份反转、动机反转、权力反转、道德反转、世界观反转、伏笔回收

## 使用方式

1. 将此 skill 安装到你的 Claude Code 环境中
2. 在对话中提供小说、文章、故事大纲或 IP 碎片
3. Skill 将自动以导演/编剧和选角/美术两个部门视角处理素材
4. 输出完整的 AI 漫剧前期制作包

## 适用场景

- 网文/IP 改编为竖屏短剧
- AI 漫画/动态漫前期分镜设计
- 短视频剧情号内容策划
- 古风/悬疑/言情/末世等多种类型
